# x-cmd Legacy Scripts

This folder contains scripts that were tied to `x-cmd` or migrated during x-cmd cleanup.

- `SETUP_X_CMD_KEYS.sh`: Original legacy x-cmd key sync script.
- `SETUP_X_CMD_KEYS_v2.sh`: Updated x-cmd-free env key + CLI checker.
- `grok-diagnostics.sh`: Legacy diagnostics script moved from top-level.
- `grok-diagnostics_v2.sh`: Updated x-cmd-free Grok/XAI diagnostics.

Run:

```bash
bash ~/scripts/x-cmd/SETUP_X_CMD_KEYS_v2.sh
bash ~/scripts/x-cmd/grok-diagnostics_v2.sh
```
